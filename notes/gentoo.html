gentoo

Gentoo is a source based Linux distribution. The handbook is a good starting point. You can get information about installing a system, working with a working system or working with portage.

The portage system is the heart of gentoo. The packages metadata is stored in /usr/portage. The sys-apps/portage package contains the emerge tool. This tool is used for portage operations.

If you want to get the recent version of portage you have to call

emerge --sync

At this point no package is updated. If you want to update the whole system you can call

emerge -uNDav world

This command updates all packages contained in the /var/lib/portage/world file.

etc files

The global configuration files are located at /etc/portage. Every build action uses the make.conf file. Changes made to this file can cause a more or less system rebuild. If you add or remove USE flags from this configuration file, it is necessary to make a system update, to apply the changes.

equery

If you want to know the package name for a file you can use equery with

$ equery b `which vim`
 * Searching for /usr/bin/vim ... 
app-editors/vim-8.0.0386 (/usr/bin/vim)

You can inspect the currently applied use flags for a specific package.

$ equery u vim
[ Legend : U - final flag setting for installation]
[        : I - package is installed with flag     ]
[ Colors : set, unset                             ]
 * Found these USE flags for app-editors/vim-8.0.0386:
